---------------------------------------
## Реализовано: 

1. Добавление книг, авторов, жанров и акций через админ-панель
2. Поиск по каталогу, фильтрация
3. Оформление заказа
4. Система отзывов (отзыв появляется на странице книги только после одобрения администратором)
5. Добавление товара в избранное (закладки)
6. Лингвистический помощник (реализован через вручное заполнение словаря в базе данных и поиск слова по этой базе)

База данных проектировалась с расчетом на добавление роли менеджера и расширение функционала интернет-магазина в будущем
